# Oral Cancer Detection using Deep Learning

## Overview
This project is an AI-powered Oral Cancer Detection system built using Deep Learning and Computer Vision techniques. The application allows users to upload oral cavity images and receive real-time predictions through an interactive Streamlit web interface.

## Features
- Upload oral cavity images
- Real-time cancer prediction
- Deep Learning-based image classification
- User-friendly Streamlit interface
- Deployed on Streamlit Cloud

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Pillow
- Streamlit
- GitHub

## Project Workflow
1. Image Upload
2. Image Preprocessing
3. CNN Model Prediction
4. Display Result
